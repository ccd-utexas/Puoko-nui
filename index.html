Instrumentation

We have developed a flexible and robust photometer for studying variable stars. The system integrates a commerical CCD camera and GPS receiver with a custom timer unit and instrument control and online data analysis software.

Two variants of the instrument exist with different cameras. The VUW system uses a Princeton Instruments MicroMax CCD. The Texas system uses the Priceton Instruments ProEM CCD. Both cameras are 1024 x 1024px back-illuminated frame-transfer CCDs. The frame transfer operation reduces the dead-time associated with digitization to a negligible level.

Our timing hardware provides precise exposure timing with a 1 millisecond resolution for exposures less than 65 seconds, or 1 second resolution for longer intervals. The unit is built around an Atmel AVR microprocessor (similar to those used in the Arduino platform) which communicates with an external GPS receiver. Source code and hardware specifications are available in the Karaka repository.

Instrument control software has been developed to run under Linux and Windows. This software is available in the Puoko-nui repository.
